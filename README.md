# Indoor CMS – Digital Signage Control Panel

Sistema de gerenciamento de mídia indoor para TVs, com autenticação, upload de imagens/vídeos, criação de playlists, agendamento de exibição e pareamento de dispositivos.

## 🚀 Tecnologias
- React + TypeScript
- Tailwind CSS
- Supabase (Autenticação, Banco de Dados, Storage)
- date-fns / dayjs
- lucide-react (ícones)

## 📋 Pré-requisitos
- Node.js 18+
- npm ou yarn
- Uma conta no Supabase (gratuita)
